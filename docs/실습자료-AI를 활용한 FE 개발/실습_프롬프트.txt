- 로그인 처리 (App.tsx)
: 이 Mock 로그인 유저 데이터를 백엔드 API 를 호출하여 적용하려고해. 
이미 로그인 백엔드 API 호출은 services/authService.ts 에 login 이라는 API 가 있으니 해당 API 를 호출하여 가져오도록 하고, 
user 정보의 타입이 백엔드 응답 데이터와 다른 부분을 맵핑하도록 해줘. 우선 로그인 처리 부분만 적용해줘

- 내 예약 목록 조회 (BookingManagement.tsx)
:이 Mock 데이터를 실제 백엔드 API 를 연동하여 적용하려고 해. 다음을 따라 적용해줘
	1.	Backend 컨트롤러 소스는 /백엔드 로컬 프로젝트 경로/car-center-devlab/src/main/java/com/carcare/domain/reservation/controller/ReservationController.java 에 있고,해당 소스의  “/my” (내 예약 목록 조회) API를 호출하여 사용자의 예약 목록을 조회하여 가져오려고 하고, 해당 API 연동만 수행해줘.
	2.	 응답 데이터 구조는 /백엔드 로컬 프로젝트 경로/car-center-devlab/src/main/java/com/carcare/domain/reservation/dto/ReservationDto.java 에 정의되어있어.
	3.	API 호출 서비스는 services/reservationService.ts 에 생성해주고,통신을 위한 기본 설정은 services/apiService.ts 를 활용해줘
	4.	응답데이터 구조는 백엔드 구조에 맞추어 types/reservation.ts 에 정의해줘.
